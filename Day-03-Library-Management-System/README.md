# Day 3 - Library Management System

## Overview

This project demonstrates intermediate SQL concepts by building a Library Management System in MySQL. It reinforces the fundamentals learned in previous projects while introducing SQL Views for simplifying frequently used queries.

The project covers database creation, table design, CRUD operations, filtering, sorting, aggregation, joins, subqueries, and views through practical business scenarios involving books and authors.

## Concepts Covered

* CREATE DATABASE
* CREATE TABLE
* INSERT
* SELECT
* WHERE
* ORDER BY
* DISTINCT
* LIKE
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* GROUP BY
* HAVING
* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* UPDATE
* DELETE
* Subqueries
* Views
