# Day 2 - Employee Management System

## Overview

This project demonstrates intermediate MySQL concepts through an Employee Management database consisting of Employees and Departments tables. It focuses on data retrieval, aggregation, joins, subqueries, and data manipulation using real-world business scenarios.

## Concepts Covered

- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Aggregate + JOIN
- UPDATE
- DELETE
- Basic Subqueries
- Correlated Subqueries

## Practice Challenges

- Find the second highest salary
- Find employees earning above their department's average salary
- Find the department with the highest average salary
- Display employees whose salary is between ₹50,000 and ₹70,000
- Find duplicate salaries
